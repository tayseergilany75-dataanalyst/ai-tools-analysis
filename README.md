# 🤖 AI Tools Analysis Dashboard

## 🎯 Project Overview
This project analyzes AI tools usage data using Power BI to understand user behavior, tool popularity, and productivity trends.

---

## 🧹 Data Cleaning
- Removed duplicates
- Handled missing values
- Corrected data types
- Created calculated measures using DAX

---

## 📈 Dashboard Features
- AI Tool Popularity Analysis
- User Behavior Analysis
- Productivity Insights
- Usage Trends
- Demographic Analysis

---

## 💡 Key Insights
- Some AI tools dominate user engagement
- Students and professionals use AI differently
- Frequent AI usage improves productivity
- Usage trends vary across age groups

---

## 🎯 Recommendations
- Promote AI awareness and training
- Improve accessibility for beginners
- Focus on productivity-oriented features

---

## 🛠️ Tools Used
- Power BI
- Excel
- DAX
- Power Query
